🌤️ React Weather App
A clean and responsive weather application built with React.js that provides real-time weather information for any city around the world.
Features

🔍 Search weather by city name
🌡️ Displays current temperature, humidity, wind speed, and weather condition
📱 Fully responsive design
⚡ Fetches live data from the OpenWeatherMap API

Tech Stack

React.js
OpenWeatherMap API
